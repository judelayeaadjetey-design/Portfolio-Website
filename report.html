Development Report

Throughout this module, I developed a portfolio website using HTML and CSS. Initially, I found structuring pages difficult, especially understanding semantic elements such as header, nav, and main. However, with practice, I improved my ability to organise content effectively. One challenge I faced was making the website responsive. Using CSS Grid and media queries helped me ensure the site worked on both desktop and mobile devices. Debugging layout issues was sometimes frustrating, but using browser developer tools helped identify problems quickly. In terms of design, I chose a simple colour scheme and readable fonts to ensure accessibility and usability. I was influenced by modern portfolio websites that focus on minimalism and clarity. I also learned the importance of version control using GitHub. Regular commits allowed me to track changes and manage my progress over time. Overall, this project improved my understanding of front-end development and helped me build confidence in creating responsive and user-friendly websites.
